6 TABLAS NORMALIZADAS:

users
├─ id (UUID PK), email (UNIQUE), password_hash
├─ name, province, phone, nif
├─ email_verified, role (customer|admin)
├─ billing_address, shipping_address (JSONB)
└─ created_at, updated_at

email_verification_tokens
├─ token (UNIQUE), user_id (FK), expires_at
└─ created_at

orders
├─ id (UUID PK), user_id (FK)
├─ items, billing_address, shipping_address (JSONB)
├─ nif, phone, subtotal, tax_amount, shipping_surcharge, total
├─ status (pending|shipped|delivered|cancelled)
├─ invoice_url, created_at, updated_at

page_view_logs
├─ url, user_agent, ip_hash (SHA256)
└─ user_id (nullable), timestamp

catalog_download_logs
├─ catalog_name, user_id (nullable), email
└─ timestamp

🚀 INSTRUCCIONES RÁPIDAS
═══════════════════════════════════════════════════════════════════════════════

1. CREAR DB:
   $env:PGPASSWORD="postgres"
   psql -U postgres -h localhost -c "CREATE DATABASE ecommerce_db;"

2. SETUP:
   cd backend && npm install && npm run migrate
   cd frontend && npm install

3. RUN:
   Terminal 1: cd backend && npm run dev    (puerto 3000)
   Terminal 2: cd frontend && npm run dev   (puerto 5173)

4. TEST:
   Abre http://localhost:5173 y registrate
